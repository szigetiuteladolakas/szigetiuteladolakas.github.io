# szigetiuteladolakas

Static one-page apartment sale website for GitHub Pages.

## Deploy on GitHub Pages

1. Create a GitHub repository named `szigetiuteladolakas.github.io`.
2. Upload everything in this folder, including the `images` directory.
3. In GitHub, open `Settings > Pages`.
4. Set the source to the main branch and root folder.
5. The site will be available at `https://szigetiuteladolakas.github.io/`.

If you later buy a custom domain such as `szigetiuteladolakas.com`, add that domain in GitHub Pages settings and create a `CNAME` file containing only the domain name.
